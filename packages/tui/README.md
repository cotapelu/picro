# @picro/tui

Terminal UI component library for pi-micro coding agent.

## Components

### Message Components
- UserMessage
- AssistantMessage
- ToolMessage
- CustomMessage
- BashExecutionMessage

### UI Components
- Modal, Toast
- ProgressBar, Stepper
- Badge, Rating
- Divider, Spacer

### Selectors
- SelectList
- SettingsList
- ModelSelector
- SessionSelector
- ThemeSelector

### Interactive
- Input
- Editor
- Markdown

## Install

```bash
npm install @picro/tui
```

## Usage

```typescript
import { Text, Box, Container } from '@picro/tui';
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
